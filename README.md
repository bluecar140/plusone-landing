# PLUSONE Landing Page

Simple marketing site for PLUSONE, built with [Next.js](https://nextjs.org) and deployed on [Vercel](https://vercel.com).

## Develop locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Push this repo to GitHub.
2. Import the repository in [Vercel](https://vercel.com/new).
3. Use the default Next.js settings (build: `next build`, output: Next.js).

No environment variables are required for the landing page.
